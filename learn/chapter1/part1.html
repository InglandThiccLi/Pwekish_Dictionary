Part 1: Alphabet of bentolRomak

Introduction

Usually, when we learn a new language, we start with the alphabet. This is also true for Pwekish. Pwekish has 2 sets of writing system, bentolRomak and bentolPwék. We will start with bentolRomak as it is easier and more similar to English.

BentolRomak consist of a set of latin letters. It is the most commonly used script due its simplicity. There are total 28 letters and each of them represents only one sound.

Fun Fact:

` ' is the only letter that uses symbol (the apostrophe) instead of latin character.
No word starts with the letter "ə". And it's also the only letter without capital form.
